Quotomania

Quotomania is a Discord bot which uses functionalities of Quotable.io to show quotes about books.

/bqotd

Provides a fresh book-related quote for the day.

/bqguess

Presents a book quote, and users have to guess which book it came from.

/bqsearch

Searches for a given book quote.

/bq-analyze

Analyzes the meaning or significance of a quote from a book.

/bq-theme [theme]

Fetches quotes from books based on specific themes (e.g., /bookquote theme courage).

/bqwallpaper

Generates an image with a book quote formatted for use as a desktop wallpaper or a social media post.

/bugreport

Report a bug.

/invite

Invite Quotomania to your server.
